# NVR
这是一个网络视频录像机软件。

`MPEG-TS` `AAC`

[English](https://github.com/sigcn/nvr/blob/main/README.md)|简体中文

### 动机
针对有 Home Server 或 NAS 等家庭长期在线设备的用户，再购买一个 NVR 硬件的话，显得不是很体面（哈哈）。而且，NVR 软件很少，开源的就更少了。这才有开源一个 NVR 软件的想法。
